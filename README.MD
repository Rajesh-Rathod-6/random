## Node.js Modules

Learned the fundamentals of the Node.js module system, including:

- **module.exports** – Export functions, objects, and variables from a module.
- **require()** – Import and use modules in other files.
- **Requiring Files** – Load local JavaScript files using relative paths.
- **Requiring Directories** – Import modules from directories using `index.js` or `package.json`.

This project demonstrates how to organize code into reusable modules for better maintainability.
